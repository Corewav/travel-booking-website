# 📌 Travel Booking Web Application

## 📖 Project Overview
The Travel Booking Web Application is a full-stack web platform that allows users to plan and manage their entire trip in one place. Users can book flights, reserve hotels, create travel itineraries, and make secure online payments.

This project is developed using the MERN Stack and focuses on providing a simple, user-friendly, and efficient travel planning experience.

---

## 🚀 Features

- User Registration & Login  
- Flight Booking System  
- Hotel Reservation  
- Travel Itinerary Planning  
- Secure Payment Gateway Integration  
- Booking Management Dashboard  
- Responsive and Clean UI

## ❗ Problem Statement

Most travel platforms focus only on one service (either flights or hotels). Users need to switch between multiple platforms, which leads to:

- Time consumption  
- Confusion  
- Poor user experience  

This project solves that by providing an all-in-one travel booking platform.

## 🏗️ System Architecture

The application follows a Three-Tier Architecture:

### 1. Presentation Layer (Frontend)
- Built using React.js  
- Handles user interaction  

### 2. Application Layer (Backend)
- Node.js + Express.js  
- Handles APIs and business logic  

### 3. Database Layer
- MongoDB  
- Stores users, bookings, and payments
